# Your Story

> 一款基于 AI 驱动的**类小说生成器**，让你以**游戏**的视角沉浸式体验小说世界。

## ✨ 项目简介

**Your Story** 是一款将传统小说阅读与游戏化体验相结合的创新项目。借助 AI 的内容生成能力，你可以：

- 🎮 **以游戏的方式阅读小说** — 不再是被动地翻页，而是亲历主角的冒险
- 🤖 **AI 实时生成剧情** — 故事不再千篇一律，每一次选择都通向不同的命运
- 🌌 **沉浸式体验** — 文字、剧情、抉择融为一体，让阅读变成一场冒险

告别旁观者视角，在这里，**你即是主角**。

## 🚀 快速开始

### 环境要求

- 操作系统：**Windows**（目前仅支持 Windows 平台）
- 无需额外安装依赖，开箱即用

### 启动步骤

1. **下载** 本仓库根目录下的 `dist.zip`
2. **解压** 压缩包到任意目录
3. 双击运行解压后的 `.exe` 文件，即可开始体验

> ⚠️ 如果系统提示安全警告，请选择「仍要运行」（项目未进行代码签名）。
